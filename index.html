IDEA 1: Interactive DJ Stand

Touch screen DJ stand with AR DJ headset + gesture recognition Genre, volume, tempo input from crowd

IDEA 2: Ultrasonic Haptic Feedback

Ultrasonic haptic feedback Make haptic holograms A ultrasonic transducer array that allows you to feel objects in mid air. Using ultrasonic waves, we can simulate tactile sensations. An AR app would show you what objects you are feeling.

IDEA 3: Interactive circuit simulation table

Interactive table for UI with Spice as a backend Place circuit components on table arranged in desired configuration, feed orientation and type of components into Spice for analysis, output result

IDEA 4: Projection based smart watch

Wearable watch/bracelet with projector on to arm with tablet/smartphone capabilities. New visual interface to exten smartphone onto a more accessible location.

IDEA 5: Sculpting Drones

A set of multiple drones which will carve a block of styrafoam to produce the object that the user draws in an AR app.
